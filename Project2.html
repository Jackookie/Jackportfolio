Project Overview

"Box can hit?" is a 2D combat game developed with Unity. The player controls a box-like character that can move left and right, and attack enemies approaching with projectiles.

Features

  • Player can move left and right using A and S keys
  • Player can attack using the "K" key
  • Attack animations using Unity's Animator Controller
  • Projectile spawner for enemies
  • Enemy projectiles follow player
  • Projectiles can be destroyed if hit by player attack

Technologies Used

Unity 6, C#
