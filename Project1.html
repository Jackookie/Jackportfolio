Project Overview

"Box can hide?" is a 3D Chasing game developed with Unity where the player controls a box that must reach the end of a line without being spotted or caught by enemy boxes.

Features

  • Enemy AI able to chase player
  • Enemy AI able to wander around
  • Player able run around using WASD
  • Player able to look around using mouse
  • Menu screen
  • Game Over Screen
  • Retry Function
  • Quit Function
  • Background Audio
  • Audio from enemy

Technologies Used

Unity 6, C#
